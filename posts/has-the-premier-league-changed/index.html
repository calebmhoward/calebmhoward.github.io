Which Premier League Big Six teams are the most clinical finishers?

Football is a game of moments. While creating chances is key, converting those opportunities into goals often separates champions from contenders. The Premier League’s “Big Six” teams are Manchester City, Liverpool, Chelsea, Manchester United, Arsenal, and Tottenham. These six have dominated the Premier League since the start, but which of these top clubs makes the most of their scoring opportunities?

Looking at the past five seasons (2020-2025), clear patterns emerge in how these teams perform in relation to their expected goals. Expected goals (xG) measures the quality of chances created based on factors like shot location, angle, and defensive pressure. By comparing xG to actual goals, we can find out which teams are clinical finishers and which squads waste chances.

Code 
library(tidyverse)
library(gt)
library(ggplot2)
library(ggalt) 
library(ggtext)

prem_data <- read_csv("premseasons.csv")

big_six <- c("Manchester Utd", "Manchester City", "Liverpool", "Chelsea", "Arsenal", "Tottenham")

last_five_seasons <- c("2020-2021", "2021-2022", "2022-2023", "2023-2024", "2024-2025")

big_six_data <- prem_data |> 
  filter(Squad %in% big_six)  |> 
  filter(Season %in% last_five_seasons)  |> 
  group_by(Squad) |> 
  summarize(
    Total_Goals = sum(Gls),
    Total_xG = sum(xG_Expected),
    Efficiency = Total_Goals / Total_xG
  ) |> 
  arrange(desc(Efficiency))

big_six_data |> 
  gt() |> 
  cols_label(
    Total_Goals = "Goals",
    Total_xG = "xG"
  ) |>
  tab_header(
    title = "Premier League Big Six Finishing Efficiency",
    subtitle = "Manchester City leads in chances created, but who's most clinical at finishing?"
  ) |> tab_style(
    style = cell_text(color = "black", weight = "bold", align = "left"),
    locations = cells_title("title")
  ) |> tab_style(
    style = cell_text(color = "black", align = "left"),
    locations = cells_title("subtitle")
  ) |>
  tab_source_note(
    source_note = md("**By:** Caleb Howard  |  **Source:** FBref")
  ) |>
  tab_style(
     locations = cells_column_labels(columns = everything()),
     style = list(
       cell_borders(sides = "bottom", weight = px(3)),
       cell_text(weight = "bold", size=12)
     ) 
   )  |>
  opt_row_striping() |> 
  opt_table_lines("none") |>
  tab_style(
    style = list(
      cell_fill(color = "#6caddf"),
      cell_text(color = "white")
      ),
    locations = cells_body(
      rows = Squad == "Manchester City")
  )

Manchester City leads in chance creation, generating a lot of high-quality chances. Their dominance in possession translates to sustained attacking pressure. This results in both high expected and actual goals. Tottenham and Arsenal follow closely behind, creating just as dangerous chances through their attack.

Code 
big_six_combined <- prem_data |> 
  filter(Squad %in% big_six) |> 
  filter(Season %in% last_five_seasons) |> 
  group_by(Squad) |> 
  summarize(
    Total_Goals = sum(Gls),
    Total_xG = sum(xG_Expected)
  )

ggplot(big_six_combined, aes(x = Total_xG, y = Total_Goals, color = Squad)) +
  geom_point(size = 5) +
  geom_abline(linetype = "dashed") +
  scale_color_manual(values = c("#ef0107", "#034694", "#dd0000", "#6caddf", "#740703", "#132257" )) +
  
  labs(
    x="xG", 
    y="Goals", 
    title="Creating and Converting Chances",

    caption="Source: FBref | By Caleb Howard"

  theme_minimal() + 
  theme(
    plot.title = element_text(size = 18, face = "bold"),

Teams like Manchester City have managed to outperform their expected goals constantly. This shows great clinical goal scorers who convert difficult chances at rates more than what is expected of them.

As well, some traditionally strong attacking teams have underperformed their xG. This suggests that either bad finishing or maybe a run of bad luck that might eventually change over a longer period of time.

Code 
big_six_combined <- prem_data |>  
  filter(Squad %in% big_six) |> 
  filter(Season %in% last_five_seasons) |> 
  group_by(Squad) |> 
  summarize(
    Actual_Goals = sum(Gls),
    Expected_Goals = sum(xG_Expected)
  )

ggplot() + 
  geom_dumbbell(data = big_six_combined, aes(y = reorder(Squad, Actual_Goals), x = Expected_Goals, xend = Actual_Goals),
    size = 2,
    colour = "grey",
    colour_x = "blue",  
    colour_xend = "red" 
  ) +
  labs(
    title = "The Gap Between Expected and Actual Goals", 
    subtitle = "If <span style='color:red;'>Actual Goals</span> are on the right of <span style='color:blue;'>Expected Goals</span>, the team overperformed.",
    caption="Source: FBref | By Caleb Howard",
    x = "Goals",
    y = "" 
  ) + 
  theme_minimal() +
  theme(
    plot.subtitle = element_textbox_simple()
  )

The efficiency gap between xGs and actual goals says something about each team’s style of play. Some like a lot of chances, while having average conversion rates. Others focus on less, higher-quality chances, relying on clinical goal scorers to get the job done.

This efficiency impacts the Premier League directly. Teams that consistently outperform their xG can get higher positions in the league than their chance creation might show, while wasteful teams may struggle to be dominate and get points.

Success in the Premier League needs both creating quality chances and the clinical ability to convert them. Teams must balance both aspects to maximize their goal production and become the best.
